Розширення для файлів для читання програмою - .txt
Для коректного зчитування інформації з файлу який містить інформацію про групи товарів, інформація про групу товарів повинна бути розділена 
символом “, ”. Кожен рядок у такому файлі відповідає одній групі товарів. 
 
Наприклад:
Меблі(Назва), Це меблі(Опис)
Солодощі, Це солодощі
 
Для файлу з інформацією про товари умови аналогічні. Слід додати тільки те, що на першому місті у рядку, 
що містить інформацію про товар, повинна стояти назва групи товарів до якої цей товар належить. 
 
Наприклад:
Солодощі(Група товарів), Цукерка(Назва), Це цукерка(Опис), Кондитерська(Назва виробника), 15(Кількість), 23.50(Ціна).
